# UniCar API

## Requisitos

* Java 21
* Git

## Configuração

Defina as variáveis de ambiente:

```text
DB_URL=
DB_USER=
DB_PASSWORD=
```

## Executando o projeto

```bash
./gradlew bootRun
```

## Swagger

Swagger UI:

```text
http://localhost:8080/swagger-ui/index.html
```

OpenAPI:

```text
http://localhost:8080/v3/api-docs
```
